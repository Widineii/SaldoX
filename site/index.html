Recursos Stack Deploy GitHub

Projeto backend em Java para portfolio

Controle financeiro com Spring Boot, JWT e dashboard responsivo.

O SaldoX simula uma aplicacao real para organizar receitas e despesas, com autenticacao, transacoes, filtros, graficos, resumo financeiro e estrutura pronta para banco PostgreSQL.

Ver codigo Como rodar
JWT Autenticacao
JPA Persistencia
Docker Deploy
H2/Postgres Banco de dados

Funcionalidades

Fluxo completo de uma aplicacao financeira

Usuarios e seguranca

Cadastro, login, JWT, Spring Security e senhas criptografadas com BCrypt.

Transacoes

Cadastro, edicao, exclusao e filtros por texto, tipo e mes.

Dashboard

Resumo de saldo, receitas, despesas, total de transacoes e graficos com Chart.js.

Arquitetura

DTOs, services, repositories, validacoes e tratamento centralizado de erros.

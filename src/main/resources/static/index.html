Sobre o projeto

SaldoX foi criado para demonstrar uma aplicação completa de portfólio.

O sistema une backend Java, banco persistente, regras de negócio e uma interface moderna. Ele simula um produto usado no dia a dia para organizar contas, parcelas, metas e relatórios.

Backend Java, Spring Boot, JPA e validações.
Dados H2 em arquivo, com login e transações salvas.
Interface Dashboard responsivo, tema escuro e relatórios.

Configurações

Local

Os dados ficam salvos em H2 local no arquivo do projeto. Para acessar o banco, abra o H2 e use JDBC URL jdbc:h2:file:./data/fintrack, usuário sa e senha vazia.

Ação sensível

Excluir transação?

Essa operação remove o registro do banco e atualiza todos os indicadores.
